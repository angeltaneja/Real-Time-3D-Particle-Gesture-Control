## 💎 Project Overview
**Kinetic Nebula** is a high-performance, browser-based generative art experience that bridges Computer Vision with WebGL. Utilizing **MediaPipe Hands** for real-time skeletal tracking and **Three.js** for particle physics, the system creates a "Neural Link" between physical human gestures and digital celestial matter.

The installation features a self-evolving ecosystem of 35,000 particles that morph through complex geometric states, responding dynamically to user proximity and hand tension.

---

## ✨ Engineering Highlights

* **Adaptive Kinetic Engine:** Real-time morphing between 8 high-fidelity templates (Sphere, Heart, Saturn, Sun, Flower, Galaxy, Butterfly, and Cherry Blossoms).
* **Gestural Interaction:** Implemented **Exponential Moving Average (EMA)** filters to smooth raw landmark data, resulting in "weighty," organic hand-following physics.
* **Super-Nova Trigger:** A custom logic-gate that detects "pinch-to-expand" velocity, triggering a white-hot luminance flash and radial expansion.
* **Parametric Geometry:** Shapes are defined via parametric and transcendental equations, mapped into $O(n)$ vertex buffers for 60FPS stability.
* **Auto-Morph Lifecycle:** An internal clock-based state machine that handles seamless transitions every 8 seconds.

## 🚀 Deployment & Local Setup

### Secure Environment Requirements
Because this project utilizes the **MediaPipe Camera Utils**, it requires a secure context (**HTTPS**) or a **Localhost** environment to run.

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/YOUR_GITHUB_USERNAME/kinetic-nebula.git](https://github.com/YOUR_GITHUB_USERNAME/kinetic-nebula.git)
    ```
2.  **Launch Local Server:**
    Open the directory in VS Code and use the **Live Server** extension to host on `http://127.0.0.1:5500`.
3.  **Production Hosting:**
    Deploy to GitHub Pages or Vercel. Ensure the "Pages" setting points to the `main` branch.
